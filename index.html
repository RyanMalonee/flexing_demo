Notes:

  • Use the browser developer tools to see the flexing.
  • The border colors don't need to be there...they're only added for your visualization
  • The column classes can be changed in value to achieve different results (flex: 8 will make the content take up 80% of the width of the div, while flex: 2 will take up 20%)
  • You can mess around with justify-content in CSS (in addition to flexing) to change the behavior of the distance between/around flexed items
  • ALL flexing should be done in the media query (there usually should not be anything with the word flex outside of the media query)
  • If you don't want the items to span from the edge of the screen to the other, use a max-width setting on the div in css
  • Go through all 4 modules here to see more information, use cases, and even more examples: https://www.w3schools.com/css/css3_flexbox.asp
  • Images and text will be automatically resized to take up the appropriate space when flexed...if you don't want this, make sure to set the max-width or max-height to a certain value in CSS
